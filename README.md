Kata Node.js

Criar uma CRUD para gerenciamento de USER

USER Model:

- firstName
- lastName
- email
- password

CRUD endpoints:

- Post
- Patch
- Delete
- GET All
- GET By Id

Recomended to use the following modules:

- express
- mongoose

Should have

- middleware for logging
  - Request Type
  - Content Type
  - Date

Bonus

- Using EJS, create a view for all users
- Using Partials
